# Implementation of Banking Database using PL/SQL.
## The following PL/SQL blocks have been implemented for the banking application-
1)	 Open a New Account
2)	Deposit Amount
3)	Withdraw Amount
4)	Amount Trans. using Account Number
5)	Check Balance (by using Account no / Mobile no)
6)	Mini statement
7)	 Account Statement
8)	View Account Details (by using Account no / Mobile no)
9)	Update Account Details 
10)	Delete Account Details (by using Account no / Mobile no)
